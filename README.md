# Image Editor

A full-stack web application for uploading, managing, editing, and downloading images. No authentication required — open-access shared image library.

---

## Features

### Image Library

- Upload multiple images at once
- Browse all uploaded images in a gallery view
- Switch between images, view metadata (dimensions, format, file size)
- Delete images

### Editing Tools

- **Crop** — define a region by x/y offset and width/height
- **Resize** — scale to a target width and/or height
- **Rotate / Flip** — rotate by degrees or flip horizontally/vertically
- **Blur** — apply gaussian blur with configurable sigma
- **Sharpen** — enhance image sharpness

### Undo / Redo

- Full edit history with visual list (newest first)
- Undo and redo any number of steps

### Download

- Export the final edited image as **JPEG**, **PNG**, or **WebP**

## How It Works

Images are stored once on disk as originals. All edits are saved as a history of operations in SQLite. Previews and downloads are generated by replaying the full edit history through Sharp — no intermediate files are created. Undo simply removes the last history entry and replays the remaining chain.

## Running the Application

### Prerequisites

- [Docker](https://docs.docker.com/get-docker/) and [Docker Compose](https://docs.docker.com/compose/install/)

### First run

```bash
docker-compose up --build
```

### Subsequent runs

```bash
docker-compose up
```

On first run it may take a minute to pull base images and install dependencies.

- **Frontend:** http://localhost:3000
- **Backend API:** http://localhost:5000/api

### Stop

```bash
docker-compose down
```

To also remove all stored images and database data:

```bash
docker-compose down -v
```

## API Overview

### Images

| Method   | Path                 | Description                  |
| -------- | -------------------- | ---------------------------- |
| `POST`   | `/api/images/upload` | Upload one or more images    |
| `GET`    | `/api/images`        | List all images              |
| `GET`    | `/api/images/:id`    | Get image metadata           |
| `DELETE` | `/api/images/:id`    | Delete image and its history |

### Editing

| Method   | Path                      | Description                                |
| -------- | ------------------------- | ------------------------------------------ |
| `POST`   | `/api/images/:id/edit`    | Apply an edit `{ action, parameters }`     |
| `GET`    | `/api/images/:id/preview` | Get current preview (full history applied) |
| `GET`    | `/api/images/:id/history` | Get ordered edit history                   |
| `DELETE` | `/api/images/:id/history` | Clear edit history                         |

### Download

| Method | Path                                              | Description           |
| ------ | ------------------------------------------------- | --------------------- |
| `GET`  | `/api/images/:id/download?format=jpeg\|png\|webp` | Download edited image |

## Known Issues

- **Sharpen** — the preview does not accurately reflect the final output; the downloaded image may look different from what is shown.
- **Crop** — currently not working.
- **Resize** — aspect ratio is not enforced; width and height are set independently. If the computed dimension is a non-integer, the result is rounded down and the image is cropped to fit, which may cut off pixels.
